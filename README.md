# 💵 Vijay A1 Tip App – C# .NET Tip Calculator

**Vijay A1 Tip App** is a simple and interactive **C# .NET application** that calculates the tip amount based on user input. This utility was developed as part of an academic assignment to demonstrate foundational skills in form-based or console-based application development using Visual Studio.

---

## 🛠️ Technologies Used

- C#
- .NET Framework or .NET Core
- Windows Forms or Console App (depending on implementation)
- Visual Studio 2022

---

## 🎯 Features

- 💰 Accepts bill amount from the user
- 📈 Calculates the tip based on standard or custom percentage
- 🧮 Displays total bill (bill + tip)
- 🧹 Option to clear/reset input fields
- 💻 Simple, clean UI/UX (for WinForms) or text-based interaction (for Console)

---

## 📁 Project Structure

VijayA1TipApp/
├── VijayA1TipApp.sln # Visual Studio solution file
├── VijayA1TipApp.csproj # C# project file (expected)
├── Program.cs # Main application logic
├── Form1.cs (if WinForms) # UI components and logic
└── bin/ & obj/ # Build outputs

---

## 🚀 How to Run

1. Open the `.sln` file in **Visual Studio 2022**
2. Build the project using `Ctrl+Shift+B`
3. Run the application using `F5` or the **Start** button
